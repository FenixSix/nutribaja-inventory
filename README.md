# NutriBaja - Sistema de Gestión de Inventario y Ventas

Sistema web moderno para gestionar el inventario y registrar ventas de productos de NutriBaja.

## Características

- ✅ **Dashboard**: Vista general de ventas, ingresos y alertas de stock
- ✅ **Gestión de Inventario**: Agregar, editar y monitorear productos
- ✅ **Registro de Ventas**: Registrar ventas y actualizar stock automáticamente
- ✅ **Historial**: Consultar todas las transacciones realizadas
- ✅ **Alertas**: Notificaciones de bajo stock

## Tecnología

- **Backend**: Node.js + Express
- **Base de Datos**: SQLite
- **Frontend**: HTML5 + CSS3 + JavaScript Vanilla

## Instalación Local

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/nutribaja.git
cd nutribaja

# Instalar dependencias
npm install

# Ejecutar el servidor
npm start
```

El servidor estará disponible en `http://localhost:3000`

## Despliegue en Render

1. Conecta tu repositorio de GitHub a Render.com
2. Crea un nuevo "Web Service"
3. Selecciona este repositorio
4. Configura:
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
5. ¡Listo! Tu aplicación estará en vivo

## Uso

1. **Dashboard**: Visualiza métricas clave de tu negocio
2. **Inventario**: Administra tus productos
3. **Registrar Venta**: Registra cada venta realizada
4. **Historial**: Consulta el registro de todas las transacciones
